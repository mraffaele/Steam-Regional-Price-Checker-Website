Games listed on Steam usually have different prices for different regions. This website allows you look up any Steam game to check the cost in most regions simultaneously. It also doubles as a home for the Chrome Extension, which performs the same instant price comparison without leaving the Steam game page.

As far as we are aware, the various regional prices comes from the publishers - so don't hate on Valve.
